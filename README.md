# AutoFinance Multibancos

Simulador de financiamento de veículos com integração a múltiplos bancos brasileiros (BV Financeira, Safra, Santander, Itaú, Bradesco, CarBank, Banco PAN, C6 Bank, GMAC/Chevrolet, Banco do Brasil, Caixa Federal, Rodobens, Porto Seguro, VW Financial e Toyota Financial).

## Funcionalidades

- Simulação de financiamento com cálculo de parcelas, taxas, CET e identificação automática da melhor oferta
- Login multiusuário com perfis de Administrador e Lojista (via Firebase Authentication)
- Histórico de simulações por usuário
- Exportação de relatórios em PDF e CSV/Excel
- Cadastro de clientes em duas etapas
- Gestão de lojistas (cadastro, credenciais bancárias por portal)
- Redefinição de senha por e-mail

## Tecnologias

- HTML, CSS e JavaScript puro (sem frameworks)
- Firebase Authentication (login e redefinição de senha)
- Firebase Firestore (banco de dados)
- Hospedagem via Netlify
